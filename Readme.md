# Lumio Chat

Lumio Chat is a real-time chat application built with the MERN stack that allows users to communicate instantly in private and group conversations.

## Features

* User Authentication (JWT)
* Real-time Messaging with Socket.io
* One-to-One Chat
* Group Chat Support
* File/Image Uploads with Cloudinary
* Online/Offline Status
* Chat History Persistence
* Secure Password Hashing
* Responsive UI

## Tech Stack

Frontend:

* React.js
* Tailwind CSS
* Axios

Backend:

* Node.js
* Express.js
* MongoDB
* Socket.io
* JWT
* Cloudinary
* Multer

## Installation

Clone the repository:

```bash
git clone <your-github-link>
```

Install dependencies:

```bash
npm install
cd client
npm install
```

Run backend:

```bash
node server.js
```

Run frontend:

```bash
npm run dev
```
